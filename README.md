# Proyectos-Python
Este repositorio está pensado para ser una biblioteca donde deje los proyectos que voy a ir haciendo para aprender python.
El objetivo es simplemente disfrutar de manera didáctica con ejercicios sencillos que poco a poco irán aumentado su dificultad. 

## Los proyectos que encontrarás
 1. Un programa que simula lanzar una moneda y ver su resultado (Cara o Cruz)
 2. Una calculadora con operaciones aritméticas básicas
 3. Un juego donde la máquina piensa un número del 1 al 10 y el usuario debe acertarlo
 4. Crear un listado de tareas por terminal y que se guarden en un .txt
 5. Organizador de Archivos
